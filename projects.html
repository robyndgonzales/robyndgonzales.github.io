Ichiberry

Multi-Page Website • 2024

01 Description

Ichiberry is a responsive multi-page website for a dessert and berry-themed café concept. Built purely with HTML and CSS, it demonstrates core front-end skills through a clean, structured, and visually appealing multi-page design.

02 Purpose

Developed to strengthen foundational front-end skills — writing semantic HTML and organized, reusable CSS. A practical exercise in building complete multi-page websites without relying on JavaScript frameworks or libraries.

  • Fully responsive layout using Flexbox and CSS Grid
  • Consistent navigation across all pages
  • Dessert product showcase with visual card components
  • Customer reviews section with custom CSS styling
  • Accessible, semantic HTML5 markup throughout
